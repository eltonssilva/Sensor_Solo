# Sensor Solo ⚙

Codigo para ESP32 para leitura do sensor de Solo HD-3002 utilizando ModBus

## How to run

1. Install project dependencies:
```bash
$ npm install
```
2. Run it:
```bash
$ npm run dev
```
3. Build:
```bash
$ npm run build
```
